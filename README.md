# Bibletree — 집중 묵상 모드 시안

바이블트리(bibletree.healthyeduministry.com) 큐티 상세 페이지를 "한 화면에 한 가지" 원칙의 단계형 흐름으로 재설계한 인터랙티브 목업입니다.

**라이브 데모**: https://fastkorea12-png.github.io/blbletree/

## 페이지

| 파일 | 내용 |
|------|------|
| `index.html` | 연령 선택 첫 화면 |
| `youth.html` | 청소년 버전 — 파랑 테마, ~해요체 |
| `young.html` | 청년 버전 — 보라 테마, ~습니다체 |
| `high.html` | 고학년 초기 시안 (참고용 · 저학년/고학년은 다른 방향의 시안 예정) |
| `records.html` | 저장한 기록을 다시 읽는 "서재 모드" 시안 — 월간 나무·타임라인·하루 상세, 종이·세리프 톤 |

예시 콘텐츠는 2026년 7월 17일 큐티(고후 1:3–7)의 실제 연령별 본문입니다.

## 묵상 흐름 (7단계)

1. **준비** — 호흡 애니메이션과 함께 마음을 모으는 도입
2. **단어** — 어려운 단어를 말씀 읽기 전에 먼저 익힘
3. **말씀** — 한 절씩 탭해서 표시해야 다음으로 진행 (대충 지나칠 수 없음)
4. **묵상** — 해설을 한 쪽씩 카드로 넘겨 읽기, 핵심 문장 강조
5. **기록** — 질문 3개를 하나씩, 질문마다 참고 구절(📖)·해설 힌트(💡) 카드 제공
6. **기도** — 호흡 아이콘 아래 기도문이 **한 문장씩** 페이드로 뜨는 집중 기도, 마지막 문장에서 "아멘 🙏"
7. **열매** — 완료 시 나무에 열매가 맺히고 연속 묵상 일수·주간 현황 표시

상단 진행 바와 단계 라벨이 항상 보여 어디까지 왔는지 알 수 있습니다.

## 설계 원칙

- 한 화면에는 한 가지 활동만 — 긴 단일 스크롤 대신 진행감 있는 단계형
- 참고 자료(구절·해설)는 필요한 지점에 다시 보여줘 앞으로 되돌아가지 않게
- 시작(준비)과 끝(기도)이 같은 호흡 아이콘으로 수미상관
- 연령별 색·어투 차등 (콘텐츠/구조는 공유)
- 라이트/다크 테마, `prefers-reduced-motion` 지원
