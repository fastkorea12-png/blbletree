# Bibletree — 집중 묵상 모드 시안

바이블트리(bibletree.healthyeduministry.com) 큐티 상세 페이지를 "한 화면에 한 가지" 원칙의 단계형 흐름으로 재설계한 인터랙티브 목업입니다.

## 흐름

준비(호흡) → 어려운 단어 → 말씀 읽기(절 탭 체크) → 묵상 해설(카드 넘김) → 묵상 기록(질문별 힌트 카드) → 기도 → 열매(완료 보상)

## 특징

- 어려운 단어를 말씀 읽기 **앞**에 배치해 읽기 전 이해를 돕습니다
- 말씀은 한 절씩 탭해서 표시해야 다음 단계로 넘어갑니다
- 묵상 질문마다 참고 구절(📖) 또는 해설 힌트(💡)를 함께 보여줍니다
- 완료 시 나무에 열매가 맺히고 연속 묵상 일수가 표시됩니다
- 라이트/다크 테마, `prefers-reduced-motion` 지원

## 실행

`index.html`을 브라우저로 열면 됩니다. 예시 콘텐츠는 2026년 7월 17일 고학년 큐티(고후 1:3–7, "위로의 하나님")입니다.
